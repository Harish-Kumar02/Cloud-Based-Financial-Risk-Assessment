# Cloud-Based Financial Risk Assessment

This repository contains a cloud-native financial risk assessment application developed as part of an academic project at the University of Surrey.  
The project focuses on using Monte Carlo simulations to support trading strategy evaluation and portfolio risk analysis through a scalable API.

The solution is designed to run efficiently in cloud environments and demonstrates practical deployment using both AWS and Google Cloud.

---

## Project Overview

This project delivers a cloud-based API for financial risk evaluation using Monte Carlo simulation techniques.  
It enables repeated scenario generation to estimate portfolio and trading strategy risk under uncertainty.

The main goals of the project are:

- Provide a scalable API for financial risk assessment
- Enable faster scenario-based evaluation for trading strategies
- Improve the reliability and performance of risk computations through cloud deployment

---

## Key Features

- Cloud-native API for financial risk and trading strategy analysis  
- Monte Carlo simulation engine for scenario-based risk estimation  
- Scalable deployment using AWS and Google Cloud  
- Optimized data processing workflows for faster execution  
- Designed for reliable and repeatable financial risk evaluations
